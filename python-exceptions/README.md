# Python - Exceptions

This project is part of the **Holberton School Higher Level Programming** curriculum.

It focuses on handling errors and exceptions in Python using `try` and `except`, without crashing the program.

## Topics Covered
- Python exceptions
- try / except blocks
- Error handling
- Safe execution of code

## Files
- `0-safe_print_list.py`  
  Prints `x` elements of a list safely using exception handling and returns the number of elements printed.

## Requirements
- Python 3
- No external modules allowed
- Followed Holberton School coding style
